---

## II. Grundsatz: Atomarität `[W]`

**Ein Modul beantwortet möglichst genau eine Frage.**

Korrekte Modulthemen: "Was ist Physik?", "Was ist Energie?", "Was ist Bayes-Statistik?"

Keine Modulthemen (→ Lernpfade): "Einführung in die Physik", "Statistik für Einsteiger"

## II.a Kleinste sinnvolle Einheit `[W]`

Ein Modul soll so klein wie möglich, aber so groß wie nötig sein. Die bevorzugte Lösung für komplexe Themen ist Aufteilung, nicht Vergrößerung.

## II.b Nummer ohne Bedeutung `[W]`

**Format:** `{DOMAIN}-L{LEVEL}-{NNNNNN}` — sechsstellig, bedeutungslos, fortlaufend innerhalb eines Levels.

## VI. Level-System L0–L9 `[W]`

| **Level** | **Bedeutung** |
|---|---|
| L0 | Kein Vorwissen erforderlich |
| L1 | Grundlagen |
| L2 | Aufbauwissen |
| L3 | Fortgeschritten |
| L4 | Spezialwissen |
| L5 | Expertenwissen |
| L6 | Frontier |
| L7–L9 | Reserviert `[OFFEN]` |

> **⚠ L-Level ≠ N-Level:** L = Modulkomplexität. N = Dokumentvoraussetzung.
