## IV. Validierungsschritte `[W]`

```bash
# V-1: Keine LRN:SSF:* mehr als id
grep -c '"id": "LRN:SSF:' kxf-learning-modules-0.1.json  # → 0

# V-2: Alle 13 Module haben legacyId
python3 -c "import json; data=json.load(open('exports/kxf-learning-modules-0.1.json')); \
  missing=[m['id'] for m in data['records']['learning_modules'] if 'legacyId' not in m]; \
  print('Fehlend:', missing or 'keine')"

# V-3: Kein learning_path-Typ mehr
python3 -c "import json; data=json.load(open('exports/kxf-learning-modules-0.1.json')); \
  lp=[m['id'] for m in data['records']['learning_modules'] if m['meta'].get('type')=='learning_path']; \
  print('learning_path:', lp or 'keine')"

# V-4: records.paths enthält 3 Einträge
python3 -c "import json; data=json.load(open('exports/kxf-learning-modules-0.1.json')); \
  print('PATHs:', len(data.get('records',{}).get('paths',[])),'(erwartet: 3)')"

# V-5: JSON valide
python3 -m json.tool exports/kxf-learning-modules-0.1.json > /dev/null && echo "JSON valide"
```
