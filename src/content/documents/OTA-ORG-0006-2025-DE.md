## I. PROJEKTGESCHICHTE

Mars Sample Return (MSR):
- Konzept: NASA/ESA-Kooperation
- Phase 1: Perseverance sammelt Proben (2021â€“2028, Jezero-Krater)
- Phase 2: Sample Return Lander holt Proben ab (~2028)
- Phase 3: Mars Ascent Vehicle bringt Proben in Orbit (~2030)
- Phase 4: Earth Return Orbiter holt Proben zur Erde (~2033)

Kosten (ursprünglich): $7 Mrd.
Kosten (revidiert 2024): $11 Mrd.
Zeitplan (ursprünglich): 2033
Zeitplan (revidiert 2024): 2040

Problem: Kosten explodieren, Zeitplan verschiebt sich.
