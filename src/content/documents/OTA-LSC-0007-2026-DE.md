## PROTOKOLL A: FASZIEN-VIBRATIONSTRAINING (8 WOCHEN) [R→T]

*Mechanotransduktion, Faszien-Remodelling, UPE-Normalisierung*

---

### A.0 Grundstruktur

- 4 Phasen à 2 Wochen: Regulation → Remodelling → Aktivierung → Deload
- 3 Sessions/Woche (Mo–Mi–Fr)
- Sessiondauer: 15–25 min
- Gesamtmechanische Progression: Frequenz × Amplitude × neuromuskuläre Komplexität

**Biphasisches Mechanohormesis-Prinzip [R]:**
Niedrige Dosis → ROS-Normalisierung → UPE ↓ oder stabil
Moderate Dosis → physiologische Aktivierung → UPE leicht ↑
Hohe Dosis → oxidativer Stress → UPE ↑↑ (unerwünscht)

### A.1 Phase 1: Hydratation & Tonusnormalisierung (Woche 1–2)

**Ziel:** Grundsubstanz-Hydratation, Mikrozirkulation, niedrige ROS-Last

| Parameter | Wert |
|-----------|------|
| Frequenz | 8–10 Hz |
| Amplitude | 0,5–1 mm |
| Beschleunigung | 0,3–0,6 g |
| Intervall | 60 s Belastung / 60 s Pause |
| Zyklen | 6–8 |
| Gesamtzeit | ~15 min |

Übungen (statisch, geringe Muskelaktivität):
- Lockerer Stand, Knie leicht gebeugt
- Unterarmstütz auf Plattform (thorakolumbale Faszie)
- Sitzende Position für Nackenfaszien (indirekt)

Erwartete UPE-Wirkung [T]: ↓ oder stabil — ROS-Normalisierung, kein Stress-Spike

### A.2 Phase 2: Faszien-Remodelling (Woche 3–4)

**Ziel:** Kollagen-Turnover, Myofibroblasten-Stimulation, Matrix-Neuausrichtung

| Parameter | Wert |
|-----------|------|
| Frequenz | 12–16 Hz |
| Amplitude | 1–2 mm |
| Beschleunigung | 0,8–1,2 g |
| Intervall | 45 s Belastung / 45 s Pause |
| Zyklen | 8–10 |
| Gesamtzeit | ~18 min |

Übungen: Isometrische Halbe Kniebeuge · Ausfallschritt statisch · Hüftscharnier-Hold ·
Scapula-Plank (Arme auf Plattform) + 3–4 s exzentrisch bei dynamischen Varianten

Erwartete UPE-Wirkung [T]: leicht ↑ (physiologische Aktivierung, kein ROS-Excess)

### A.3 Phase 3: Neuromuskuläre Aktivierung (Woche 5–6)

**Ziel:** Kraftentwicklung, elastische Rückstellfähigkeit, kurzzeitiger Mito-Boost

| Parameter | Wert |
|-----------|------|
| Frequenz | 25–30 Hz |
| Amplitude | 2–3 mm |
| Beschleunigung | 2–3 g |
| Intervall | 30–45 s Belastung / 60 s Pause |
| Zyklen | 6–8 |
| Gesamtzeit | ~20 min |

Übungen: Isometrische tiefe Kniebeuge · Einbeiniger Stand · Mini-Squats ·
Schnelle Wadenfedern · Push-up-Hold auf Plattform

**Wichtig:** Keine zwei aufeinanderfolgenden Hochfrequenz-Tage.

Erwartete UPE-Wirkung [T]: akut ↑ — sollte 24–48 h normalisieren

### A.4 Phase 4: Deload & Integration (Woche 7–8)

**Ziel:** Struktur konsolidieren, ROS normalisieren, Überlastung vermeiden

| Parameter | Wert |
|-----------|------|
| Frequenz | 10–14 Hz |
| Amplitude | 1–1,5 mm |
| Beschleunigung | 0,6–1 g |
| Intervall | 60 s Belastung / 60 s Pause |
| Zyklen | 6–8 |
| Gesamtzeit | ~15–18 min |

Übungen: CARS auf Plattform · Leichte isometrische Holds · Atemfokus (parasympathisch)

Erwartete UPE-Wirkung [T]: ↓ Richtung Ausgangswert

### A.5 Regionale Schwerpunktsetzung (Rotationsprinzip)

| Tag | Faszienschwerpunkt |
|-----|-------------------|
| A | Plantarfaszie + hintere Kette |
| B | Thorakolumbalfaszie + Gluteal |
| C | Ventrale Linie + Schultergürtel |

Verhindert lokale Überdosierung der mechanischen Energie.

### A.6 Monitoring-Parameter

Zwischen Sessions prüfen: Morgen-HRV · Lokale Druckschmerzschwelle · Subjektive Steifigkeit 0–10

Intensität reduzieren, wenn: Steifigkeit >2 Tage ansteigt · HRV deutlich fällt · Druckschmerz zunimmt

### A.7 Λ-FSS-Anpassung [S→F]

> **[KN-A.1] FSS-Träger brauchen modifizierte Phase 3 [S]**
>
> Bei FSS-III+-Trägern ist Phase 3 (25–30 Hz, 2–3 g) das kritische Fenster.
> Der ROS-Spike von Phase 3 kann bei unvorbereitetem Λ-System DS-Grad-I-Symptome
> auslösen — nicht durch Erschöpfung, sondern durch UPE-Kohärenz-Verlust.
>
> EIQM-Anpassung 2031: Phase 3 bei FSS-III+ auf 20–25 Hz, Amplitude 1,5–2 mm;
> gleichzeitig Vorbehandlung mit Resonanzatmung (6/min, 10 min) vor der Session
> und CoQ10 200 mg 2h vorher.
>
> — T.P.K.

## PROTOKOLL B: PHOTOBIOMODULATION (8 WOCHEN) [R→T→S]

*Optische Mito-Modulation, Redox-Homöostase, Gewebsregeneration*

---

### B.0 Grundparameter [R]

| Parameter | Bereich |
|-----------|---------|
| Rot | 630–680 nm (oberflächlich, Haut/Faszien) |
| NIR | 780–880 nm (tief, Mito/Muskel/ZNS) |
| Irradianz | 10–60 mW/cm² |
| Fluenz | 3–30 J/cm² (nie über 40 J/cm²) |
| Frequenz | 3×/Woche (Basis), 2×/Woche (Deload), bis 4× (Peak) |
| Sessiondauer | 10–20 min/Region |

**Biphasische Dosis-Antwort:** Überdosierung hemmt mitochondriale Funktion.
Warnsignale: Müdigkeit nach Licht · erhöhte Druckempfindlichkeit · Schlafverschlechterung
→ Fluenz um 30–50% reduzieren.

### B.1 Phase 1: Anti-inflammatorisch / Redox-Normalisierung (Woche 1–2)

**Parameter:**

| Wert | Spezifikation |
|------|---------------|
| Wellenlänge | 660 nm + 810 nm kombiniert |
| Irradianz | 10–20 mW/cm² |
| Fluenz | 4–6 J/cm² pro Areal |
| Zeit | ~3–5 min/Areal |
| Modus | Kontinuierlich |
| Timing | Nach Vibrationsprotokoll (regenerativ) |

Applikationszonen: Schmerzhafte Faszienpunkte · Thorakolumbaler Übergang · Plantarfaszie

Erwartete UPE-Wirkung [T]: ↓ (NO-Freisetzung ↑, ROS ↓)

### B.2 Phase 2: Mitochondriale Aktivierung (Woche 3–4)

| Wert | Spezifikation |
|------|---------------|
| Wellenlänge | 660 nm + 810–830 nm |
| Irradianz | 20–40 mW/cm² |
| Fluenz | 8–12 J/cm² |
| Zeit | 4–6 min/Areal |
| Timing | 30 min vor Vibration (Remodelling-Synergie) |

Zielgewebe: Myofasziale Übergänge · Triggerpunkte · Sehnen-Faszien-Interfaces

Erwartete UPE-Wirkung [T]: moderate adaptive ↑ (Cytochrom-c-Oxidase Aktivierung)

### B.3 Phase 3: Remodelling & Leistungsintegration (Woche 5–6)

| Wert | Spezifikation |
|------|---------------|
| Wellenlänge | Schwerpunkt 810–880 nm |
| Irradianz | 30–60 mW/cm² |
| Fluenz | 12–20 J/cm² (lokal bis 30 J/cm²) |
| Zeit | 5–8 min/Areal |
| Optional | Gepulst 10–40 Hz (neuromodulatorisch) |
| Timing | Vor Vibration (20–40 min Abstand) |

Erwartete UPE-Wirkung [T]: akut ↑, normalisiert <24 h

### B.4 Phase 4: Konsolidierung (Woche 7–8)

| Wert | Spezifikation |
|------|---------------|
| Wellenlänge | 660 + 810 nm kombiniert |
| Irradianz | 15–25 mW/cm² |
| Fluenz | 6–10 J/cm² |
| Optional | Gepulst 5–10 Hz (vagal orientiert) |
| Timing | Nach Vibration oder Off-Days |

Erwartete UPE-Wirkung [T]: ↓ Richtung Baseline

### B.5 Λ-FSS-spezifische Anpassungen [S→F]

> **[KN-B.1] Kopf-NIR bei FSS-III+ kritisch [S]**
>
> Standard-PBM-Protokolle für den Schädel/ZNS-Bereich sind bei FSS-III+-Trägern
> mit erhöhter Vorsicht anzuwenden: NIR auf Kopf kann UPE-Kohärenz kurzfristig
> destabilisieren und DS-Grad-I-Symptome auslösen.
>
> EIQM-Anpassung 2031: NIR-Exposition primär auf Nacken/Herzbereich und große
> Muskelgruppen, nicht direkt auf Schädeldach. Kopf-UPE vor und nach Session messen.
> Immer mit Atemprotokoll koppeln (6/min, 5 min post-PBM).
>
> — T.P.K.

## PROTOKOLL C: INTEGRATIVES 8-WOCHEN-PROGRAMM [R→T→S/F]

*Bewegung · Licht · Akustik · Regulation als Gesamtsystem*

---

### C.0 Grundstruktur

- Rhythmus: 3+1+3+1 (3 Belastungswochen + 1 Deload, zweimal)
- 5 aktive Tage/Woche, 2 Off/Minimal-Tage
- ~60–90 min/Tag (aufteilbar auf 4 Module)

**Module:**
1. Bewegung (Mito-/Faszien-Block)
2. Licht (PBM / Habitatbeleuchtung)
3. Akustik (Frequenz-/Resonanzblock)
4. Regulation (Atmung, Schlaffenster)

### C.1 Wochen 1–2: Basis & Regulation

**Ziel:** Redox beruhigen, Rhythmus herstellen, Faszien/Herz-Hirn-Kohärenz etablieren.

| Modul | Intervention |
|-------|-------------|
| Bewegung | 3× Zone 1–2 Ausdauer (20–30 min) + 2× Faszien-Mobility (leichte Schwingungen, 5–10 min Federn) |
| Licht | Morgens 20–30 min Vollspektrum warm · Abends ≤2700 K · Kein gezieltes PBM |
| Akustik | 10–15 min/Tag ruhige Klangumgebung (60–80 bpm) · Keine Resonanzfrequenz-Protokolle |
| Regulation | 2× tägl. Atemprotokoll (5:5 s, 6 Züge/min, 5–10 min) · Fester Schlafrhythmus (±30 min) |

### C.2 Wochen 3–4: Aktivierung & erste Resonanz

**Ziel:** Mito-Output erhöhen, UPE-Responsivität testen.

| Modul | Intervention |
|-------|-------------|
| Bewegung | 3× Ausdauer mit 3–4 leichten Intervallen (30–60 s) + 2× Kraft/Zirkel (RIR 2–3 athletisch / KG + Bänder therapeutisch) |
| Licht (3×/Woche) | 5–10 min Rot/NIR lokal (Muskelgruppen, oberer Rücken) · Immer nach Training oder abends |
| Akustik (3×/Woche) | 2× Resonanzband A (147–175 Hz dominant) + 1× Resonanzband B (420–440 Hz harmonisch) · Subjektives Log führen |
| Regulation | Atemprotokoll auf 10–15 min/Tag · 1 aktive Regenerationseinheit/Woche |

### C.3 Woche 5: Deload & Integration

| Modul | Intervention |
|-------|-------------|
| Bewegung | 3 lockere Einheiten (keine Intervalle) · mehr Faszien-/Dehnungsfokus |
| Licht | 2×/Woche, halbe Dosis |
| Akustik | Nur ruhige Frequenzumgebung · Keine Ziel-Frequenz-Protokolle |
| Regulation | Optional 1× längere Session (30–40 min) Meditation / Body Scan |

### C.4 Wochen 6–7: Zielgerichtete Leistungsphase

**Ziel:** Mito-Output/Resilienz hochfahren, Λ-Resonanzfähigkeit bewusst nutzen.

| Modul | Intervention |
|-------|-------------|
| Bewegung | 2–3× Ausdauer mit strukturierten Intervallen + 2× Kraft (RIR 1–2 athletisch / moderat therapeutisch) |
| Licht (3–4×/Woche) | 1–2× vor/nach Training + 1–2× regenerativ Off-Days · Schlafproblem → PBM nur tagsüber |
| Akustik (3–4×/Woche) | 2× "Power-Resonanz" (147–175 Hz, rhythmisch, vor Training) + 1–2× "Kohärenz-Resonanz" (420–440 Hz weich, bei Atem-/Meditationssession) |
| Regulation | Atemarbeit vor/nach Belastung · Schlafhygiene strikt (kein helles Blau abends) |

### C.5 Woche 8: Abschluss & Konsolidierung

| Modul | Intervention |
|-------|-------------|
| Bewegung | Umfang −30–40% vs. Woche 6/7 · Optional: Testwiederholung von Woche 1 (Vergleich) |
| Licht | 2–3 leichte Sessions |
| Akustik | 2–3 Sessions mit am besten verträglichen Frequenzbändern |
| Auswertung | Schlaf · Energie · Stimmung · Belastbarkeit · FSS-/Stresssymptome |

> **[KN-C.1] OTA-Auswertungsslot [S→F]**
>
