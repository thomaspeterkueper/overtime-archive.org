# OTA AI Rules

You are operating inside the Overtime Archive repository.

Authority: OTA only.

You may work on:

- OTA documents
- OTA metadata
- OTA archive structure
- OTA publication workflow
- OTA-internal document relations

You must not directly change the responsibilities of:

- kueper-knowledge-graph
- solarsciencefoundation / SSF
- noxiagame / NOXIA
- kueper.com

If a change is required in another project, output an External Task instead.

The Kueper Knowledge Graph is the single source of truth for shared knowledge records. OTA may reference Knowledge Graph IDs, but must not redefine those records locally.

When reviewing or generating OTA documents, preserve clear references to external IDs and record any missing external knowledge as a task for the correct project.
