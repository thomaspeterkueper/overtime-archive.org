# Contributing to Overtime Archive

This repository is responsible for OTA only.

Before changing files, check the scope.

Allowed in this repository:

- OTA documents
- OTA metadata
- OTA archive structure
- OTA publication workflow
- OTA-internal document relations

Not owned by this repository:

- kueper-knowledge-graph
- solarsciencefoundation / SSF
- noxiagame / NOXIA
- kueper.com

When a change is needed in one of those projects, write an External Task instead of changing the other project here.

An External Task should include:

- target project
- priority
- task type
- source OTA document or issue
- short summary
- reason
- proposed change

OTA may reference Knowledge Graph IDs. OTA must not redefine Knowledge Graph records locally.
